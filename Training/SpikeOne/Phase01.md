# Phase 1 — Get Something on the Screen

**Goal:** Build a React app with 3 widgets using plain CSS. No abstractions yet.

---

## What to Do

- Scaffold with Vite — `npm create vite@latest` (already done in `training/`)
- Build 3 independent widgets: a **counter**, a **text display**, and a **form input**
- Style with a single flat CSS file — no classes shared between components

The Vite scaffold in the `training/` directory is your starting point. Strip out the default template content in `App.tsx` and replace it with your three widgets.

---

## Why This Matters

In real projects, the first sprint is about proving the concept works at all.
Clean code comes later. Get it rendering.
