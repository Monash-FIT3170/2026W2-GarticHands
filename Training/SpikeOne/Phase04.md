# Phase 4 — Add Tailwind

**Goal:** Restyle everything using Tailwind utility classes.

---

## What to Do

- Install and configure Tailwind
- Style the `Card` component first with base utilities — `rounded`, `shadow`, `p-4`, `border`
- Style each widget individually on top of the card base
- Avoid writing any custom CSS — if you reach for it, there's probably a Tailwind class

---

## Why This Matters

Tailwind is the dominant CSS approach in modern teams.
The mental shift is from "write styles" to "compose utilities." It feels awkward at first and then clicks.
